Code

This page contains links to code for some recent projects from the Yu group (see specific papers for other code repositories).

  • ACD: pytorch implementation providing hierarchical interpretaions for DNN predictions

  • iRF: an implementation of iterative Random Forests in python based on scikit-learn

  • Superheat: An r package for generating beautiful and customizable heatmaps

  • CD: pytorch implementation providing scores for features and groups of features for DNN predictions
